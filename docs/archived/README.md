# 📦 Archived 文件區

本目錄存放已過時、不再使用或已被新文件取代的舊文件。

## 📋 文件歸檔原則

### 何時歸檔文件

1. **文件已被新版本取代** - 新文件已建立，舊文件保留作為參考
2. **文件內容已過時** - 文件中的資訊不再符合當前專案狀態
3. **文件結構已重組** - 文件內容已整合到新的文件體系中
4. **文件不再需要** - 功能已移除或不再使用

### 歸檔流程

1. 將文件移至 `docs/archived/` 目錄
2. 在文件開頭加入歸檔說明（為什麼歸檔、替代文件位置）
3. 更新所有引用該文件的連結
4. 在本 README 中記錄歸檔資訊
